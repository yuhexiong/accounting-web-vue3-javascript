# Accounting Web

## Overview

- Language: JavaScript
- Web FrameWork: Vue3

## ENV

copy .env.example and rename as .env

```bash
VUE_APP_API_BASE_URL=http://localhost:7777
```

## Run

### install dependencies

```bash

npm install

```

### linting code

```bash

npm run lint --fix

```

### build

```bash

npm run build

```

### run

```bash

npm run serve

```
